# marvel-app :iphone:

## Architecture

**Marvel App** was developed using the **MVVM-C** architecture

MVVM-C:

**M**odel -> Represents data<p>
**V**iew -> Displays data<p>
**V**iew**M**odel -> Bussiness Logic and Data Binding<p>
**C**oordinator -> Navigation between screens<p>
